Explain control flow and loops using an example process from everyday life.

The control flow is the order in which the computer executes statements in a script.

Code is executed in order from the first line in the file to the last line, unless the computer runs across the structures that change the control flow, such as conditionals and loops. An example might be some code that is used to gather user data from a fillout form on a website. The script submits validated data, but if the user makes an error that is detected or accidentally leaves a required field empty, the script prompts them to fill it in. To do this, the script uses a conditional structure or if...else, so that different code executes depending on whether the form is complete or not.

Describe what the DOM is and an example of how you might interact with it.

The DOM (Document Object Model) is a browsers own internal programatic representation of the web page. This means that you can go into the DOM and manipulate the browsers representation of the HTML document to change various elements (objects) on a webpage without changing the source HTML.

An example of this might be changing the "h1" tag to a different text. This change is only be temporary and will be lost of the the page is refreshed in the browser. You can also interact with the DOM via JavaScript, for example using the "document.getElementById(ID)" statement.



Explain the difference between accessing data from arrays and objects.

Searching an Array for a Value

An array is a set of ordered values, or collection of data that is represented by a container of square brackets "[]"

When looking to find the position of a value within an array we can use the JavaScript console to search via arr.indexOf(). This calculates the position of a value within the array starting at 0.

Accessing Object Data

An object is simply a container for properties of JavaScript data. An object is represented within curly brackets " { } ". Data that is listed in these brackets are properties that are given values. An example for an object may be "Car". A cars object values may be:

  • car.name = Prius
  • car.weight = 1000kg
  • car.colour = Blue

You can access an objects properites with either of the following two syntax

  1. objectName.propertyName
  2. objectName["propertyName"]
